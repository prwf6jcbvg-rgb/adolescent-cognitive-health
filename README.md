# Lifestyle Factors and Cognitive Health in Adolescents

## Overview
This project explores how lifestyle factors such as sleep duration,
physical activity, and screen time are associated with cognitive
performance indicators in adolescents.

## Motivation
Adolescent cognitive health is influenced by modifiable lifestyle
factors. Understanding these relationships can inform public health
education and prevention-focused approaches.

## Methods (Planned)
- Data cleaning and exploratory analysis in Python
- Statistical analysis of lifestyle variables
- Visualization of trends and associations

## Ethics
This project uses publicly available or anonymized data only.
No diagnosis or personal identification is involved.

## Status
